# falling_sand_sim
WIP falling sand simulation made with rust and notan. Currently trying to make this my best attempt at it.

Completed tasks:
* Base physics
* Basic elements
* Chunking system
* Dirty rectangle system
* Other optimizations (such as disabling and enabling chunks)
* Multithreading
